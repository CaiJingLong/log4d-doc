# Server

## install

before install you need see [pub global](https://www.dartlang.org/tools/pub/cmd/pub-global)

```bash
pub global activate log4d
```

## Usage

```bash
log4d
```


```bash
$ log4d -h
-o, --output          output path
-p, --port            server port
                      (defaults to "8899")

-h, --[no-]help       usage
-c, --[no-]console    show log in console
                      (defaults to on)
```
